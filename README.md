# atamaita

> 簡易頭痛記録アプリ


## 本アプリの価値
巷の頭痛記録アプリは多機能過ぎて学習コストが高く、入力項目が多すぎる。

出来る限り手軽に素早く頭痛記録出来るアプリを作る。

## 開発目的
- AWS Amplify、GraphQL、Nuxtを学ぶ。
- Githubを使った開発の進め方に慣れる。

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
